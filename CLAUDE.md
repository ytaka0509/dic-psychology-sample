# 心理学用語集サイト — CLAUDE.md

## プロジェクト概要

心理学の重要概念を学べる用語集サイト。
3つのカテゴリ（認知／感情／行動）に分かれた静的サイト。
JavaScriptデータファイルからデータを読み込み、カテゴリフィルター付きの用語一覧を表示する。

**管理方針**: ローカル管理のみ。GitHubへのプッシュは行わない。

---

## ディレクトリ構成

```
/
├── index.html          # トップページ（ジャンル選択）
├── cognition.html      # 認知（Cognition）用語集ページ
├── emotion.html        # 感情（Emotion）用語集ページ
├── behavior.html       # 行動（Behavior）用語集ページ

│   └── style.css       # 共通スタイル（淡いグリーンテーマ）

│   ├── glossary.js     # フィルター・レンダリング共通ロジック
│   └── index-search.js # トップページ横断検索ロジック
└── data/
    ├── cognition-data.js  # 認知用語データ（~50件）
    ├── emotion-data.js    # 感情用語データ（~50件）
    └── behavior-data.js   # 行動用語データ（~50件）
```

## データファイル形式

各データファイルは以下のJS形式で作成する。

```js
const COGNITION_DATA = [
  {
    category: "バイアス",
    term: "確証バイアス",
    desc: "自分の信念を支持する情報ばかり集め、反する情報を無視する傾向。",
    detail: "詳細説明テキスト（省略可）"
  },
  ...
];
```

- `category`: カテゴリ名（フィルタリングに使用）
- `term`: 用語名
- `desc`: 短い説明文
- `detail`: 詳細説明（省略可、省略時はモーダル非表示）

## データファイル一覧と収録カテゴリ

| ファイル | ジャンル | カテゴリ | 件数 |
|---|---|---|---|
| `data/cognition-data.js` | 認知 | バイアス／記憶／判断・意思決定／注意・知覚 | ~50件 |
| `data/emotion-data.js` | 感情 | ストレス・感情調整／モチベーション／不安・恐怖／ウェルビーイング | ~50件 |
| `data/behavior-data.js` | 行動 | 行動心理・条件づけ／習慣形成／社会心理／コミュニケーション | ~50件 |

## 各ページの仕様

### トップページ（index.html）
- 3つのジャンルカードを横並びで表示
- 各カードはジャンル名・説明・用語数を表示
- カードクリックで対応ページへ遷移
- 全ジャンル横断検索機能あり

### 用語集ページ（cognition.html / emotion.html / behavior.html）
- ページロード時にデータファイルを読み込む
- カテゴリフィルターボタンを上部に表示（「すべて」＋カテゴリ名を動的生成）
- 用語はカード形式で一覧表示
- フィルターボタンクリックで対象カテゴリのカードのみ表示
- 検索ボックスで用語名・説明をリアルタイム絞り込み

## CSS 設計方針

- CSS変数でカラーテーマを管理（`:root` にまとめる）
- ベースカラー：淡いグリーン（`#f0fdf4`）
- アクセントカラー：グリーン系（`#16a34a`）
- レスポンシブ対応：フィルターボタンは横スクロール、カードはグリッドレイアウト
- フォントはシステムフォント（日本語対応）

```css
:root {
  --green:       #16a34a;
  --green-dark:  #14532d;
  --green-light: #f0fdf4;
  --green-mid:   #bbf7d0;
  --bg:          #f0fdf4;
  --surface:     #ffffff;
  --ink:         #1c2b1e;
  --muted:       #5a7a62;
  --border:      #d1fae5;
}
```

## 実装時の注意事項

- ローカルで `fetch` は使用せず、JSデータファイルをscriptタグで読み込む方式
- ローカルファイル（`file://`）でも動作可能
- 新しいジャンルを追加する場合：
  1. `data/` に新データJSファイルを追加
  2. HTMLページを1つ追加（既存ページをコピーして内容変更）
  3. `index.html` のジャンルカードに追記
  4. `js/index-search.js` の `ALL` 配列に追加
- 用語の追加・編集はデータJSファイルのみを変更すればよい
